자연어로 입력된 할 일을 구조화된 데이터로 변환하는 기능을 구현해 주세요.

1. 구현 조건
- 새할 일 추가 창에 AI 기반 할 일 기능 생성 연결
- Gemini API (genemi-2.5-flash)를 AI SDK 로 연동
- 환경 변수 GOOGLE_GENERATIVE_AI_API_KEY 사용
- Next.js API Route로 백엔드 구현
- 반환 결과는 기존 할 일 데이터 구조(todos 테이블 스키마)와 일치
- 날짜와 시간은 정확히 추출하고, 시간이 누락된 경우 기본 값(예 09:00) 설정
- 문맥을 기반으로 우선 순위 자동 판단
- 오류 처리 및 입력 검증 로직을 포함해 안정적으로 동작하도록 설계

2. 입력 예시
내일 오후 3시까지 중요한 팀 회의 준비하기

3. 출력 예시
{
    title: "팀회의 준비",
    due_date: "2026-01-16",
    due_time: "14:00",
    priority: "high",
    category: "업무"
}
